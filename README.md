# experiments-desings-I
# 📦 Instrucciones para abrir el proyecto en RStudio

## 1. Descargar y descomprimir

* Descargue el archivo `.zip` entregado.
* Extraiga su contenido en una carpeta de su preferencia (por ejemplo, en el escritorio).

---

## 2. Abrir el proyecto en RStudio

* Inicie **RStudio**.
* Haga clic en:

  * `File` → `Open Project`
* Busque la carpeta descomprimida.
* Seleccione el archivo con extensión `.Rproj`.

---

## 3. Verificar el entorno de trabajo

Una vez abierto el proyecto:

* RStudio automáticamente usará la carpeta del proyecto como directorio de trabajo.
* Puede verificarlo ejecutando en la consola:

```r
getwd()
```

---

## 4. Ejecutar el código

* Abra el archivo `.R` correspondiente (script principal).
* Ejecute el código línea por línea con:

  * `Ctrl + Enter` (Windows)
  * `Cmd + Enter` (Mac)

---

## 5. Notas importantes

* Todos los archivos necesarios (datos, scripts, etc.) están incluidos en la misma carpeta del proyecto.
* No es necesario modificar rutas de archivos.
* Si algún archivo no se carga correctamente, verificar que esté en la misma carpeta que el `.Rproj`.

---

## ✅ Recomendación

Para evitar errores, siempre abra primero el archivo `.Rproj` antes de ejecutar cualquier script.
